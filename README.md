# Reuse Incentives Working Group

A collaboration between [Continuous Science Foundation](https://continuousfoundation.org) and [PREreview](https://prereview.org) exploring how peer review can evolve in a modular publishing ecosystem.

The **Reuse Incentives Working Group** is a collaboration between the [Continuous Science Foundation](https://continuousfoundation.org) and [Creative Commons (CC)](https://creativecommons.org). The group brings together participants from across the research ecosystem to explore how licensing can function as infrastructure that enables the reuse of research outputs in an increasingly modular research environment.

## How to contribute

Open a pull request. Edit the content, suggest changes, or add session notes. All outcomes are licensed CC-BY-4.0.

## Working locally

This site is built with [MyST](https://mystmd.org). To preview locally:

```bash
cd content/working-group-2026
```
